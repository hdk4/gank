# gank

A gank client based on vue.

## Technology Stack

- [vue2](https://vuejs.org/)
- [vue-cli3](https://cli.vuejs.org/)
- [vue-router](https://router.vuejs.org/)
- [vuex](https://vuex.vuejs.org/)
- [axios](https://github.com/axios/axios)
- [webpack](https://webpack.js.org/)

## Special Thanks To

- [gank](https://gank.io/)

## Setup

```
yarn install
```

## Run

```
yarn run dev
```

or

```
yarn run serve
```

## Build

```
yarn run build
```
